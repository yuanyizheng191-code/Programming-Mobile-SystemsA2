# Inventory Management System
A simple, clean, responsive inventory application built with Angular.

## Features
- Add, edit, delete products
- Mark products as Popular
- Stock status: In Stock / Low Stock / Out of Stock
- Search by ID, Name, Category
- Modern English UI
- Local data storage

## Pages
- Home: Dashboard
- Inventory: Manage items
- Search: Find items
- Help: User guide
- Privacy: Privacy policy

## Tech Stack
- Angular
- TypeScript
- HTML
- CSS

## How to Run
1. npm install
2. ng serve --open
3. Visit http://localhost:4200

## Usage
1. Go to Inventory page
2. Fill product details
3. Click Save
4. Edit or delete items as needed
5. Search items in Search page

## Notes
- All data stored locally
- No data sent to cloud
- Clean, professional layout
